Git Essentials

Practicalities, beyond syntax.

Setting things up

1. Check whether Git is present ...

git version

2. Set up your local Git installation.

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com
git config --global -l --show-origin

3. List the contents (for kicks & giggles).

cat C:/Users/johndoe/.gitconfig

Personalization

1. Set up diff & merge tool

git config --global diff.tool bc
git config --global difftool.bc.path ".../BComp.exe"
git config --global merge.tool bc
git config --global mergetool.bc.path ".../BComp.exe"

Getting a sample project

1. Clone the repository

git clone https://github.com/liveweird/WebSlides.git

2. Learn about the remote

git remote
git remote get-url origin

3. Learn about the branch(es)

git branch
git branch -r
git branch snafu
git checkout snafu
git branch -d snafu
